# Customer-Data-Analysis
An end-to-end customer shopping behavior analysis project using Python, PostgreSQL, and Power BI to uncover customer insights, spending patterns, and business recommendations.

# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, covering data loading, exploration, cleaning, SQL analysis, dashboard development, and business reporting. The objective is to transform raw data into meaningful insights using industry-standard analytics tools.

---

## Dataset

The project uses a structured dataset containing business-related information for analysis. The dataset is loaded into Python for preprocessing and exploratory data analysis before being imported into SQL databases for querying.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Performed initial inspection of data structure and quality.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships.
* Identified trends, patterns, and potential issues.
* Created visualizations to support findings.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected data types.
* Standardized and prepared the dataset for analysis.

### 4. SQL Analysis

* Imported cleaned data into relational databases.
* Executed SQL queries using:

  * PostgreSQL
  * MySQL
  * SQL Server
* Performed filtering, aggregation, joins, grouping, and business-focused analysis.

### 5. Dashboard Development

* Built an interactive Power BI dashboard.
* Included KPIs, charts, filters, and slicers.
* Enabled easy exploration of business insights.

### 6. Reporting

* Prepared a summary report highlighting key findings.
* Created a professional presentation using Gamma.

## How to Run

1. Clone the repository.

```bash

```

2. Install the required Python libraries.

```bash
pip install -r requirements.txt
```

3. Open the Jupyter Notebook and run the analysis.


4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server and execute the SQL scripts.

5. Open the Power BI dashboard (`.pbix`) to explore the visualizations.

6. Review the report and presentation for project insights.
